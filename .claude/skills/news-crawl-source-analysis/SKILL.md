# 新闻来源分析

## 目标
为新增或失败来源生成稳定、可直接实现的提取方案。

## 步骤
1. 确认输入：来源 URL、语言、目标字段、更新频率。
2. 在 `src/main/tasks/new_tasks/` 或 `src/main/tasks/secondary_tasks/` 中寻找最接近参考。
3. 构建字段映射：
   - 列表页：URL、标题、发布时间、图片
   - 详情页：标题、正文、发布时间、首图
4. 定义等待策略与重试条件。
5. 标注风险：反爬、动态渲染、日期变体、分页变体。
6. 提供主选择器与备选选择器。
7. 若涉及正文图片展示，补充：
   - 正文根节点选择器（用于保留图文顺序）
   - 正文图片选择器（src/data-src/data-original）
   - 占位输出格式（默认 Markdown 图片语法）
